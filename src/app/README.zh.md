# App 占位

Next.js 应用放在这里。首个实现目标：

- `/` dashboard：连接钱包、创建 AgentSpace、列出记忆/工件/日志。
- `/api/spaces`：创建/列出 spaces。
- `/api/spaces/[id]/memories`：写入/读取记忆。
- `/api/spaces/[id]/context`：组装活跃上下文。
- `/api/spaces/[id]/artifacts`：上传/列出工件。
- `/api/spaces/[id]/proof-logs`：追加/列出执行证明日志。
